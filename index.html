  • git init - "Inicijacija git - a"
  • git config user.name "Ime Prezime"
  • git config user.mail "e-maiil@gmail.com"
  • git status - (sta se trenutno desava iu nasem projektu)
  • git add .index.html - (koji fajl zelimo) - Staging Area
  • git add . - (. oznacava sve fajlove u projektu) - Staging Area
  • git commit -m "Poruka" (da sacuvamo promene - trenutni snimak projekta)
  • git log - izbacuje sve promene
  • git log -1 - izbacuje poslednju promenu
  • git log -p izbacuje razlike - u kom commitu su nastale promene
  • git log -pretty=oneline (infirmacije u jednoj liniji)
  • git log --pretty=format:"%s" - izbacuje samo poruke na promenama
  • git log --pretty=format:"%s %an" - poruke i ko je autor promene.... moze i po datumu, useru, samoj promeni ako je function
